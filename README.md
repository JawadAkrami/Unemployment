

# Unemployment Analysis & Dashboard 

## Overview

This project analyzes U.S. unemployment trends from 1990 to 2024 using SQL and an interactive dashboard. It focuses on how unemployment changes over time and across different geographic levels.

---

## Data Preparation

* Cleaned and standardized column names
* Checked for duplicates and missing values
* Validated unemployment rates
* Considered labor force size to reduce small-area volatility

---

## Key Insights

* **Major spikes in 2009–2011 and 2020**, with 2020 being the largest
* **Strong recovery in 2021–2022**
* **States are the most stable**, while smaller areas are more volatile
* **Counties generally have higher unemployment rates**

---

## Dashboard

The dashboard highlights:

* Total unemployment and employment trends
* Average unemployment rate over time
* Clear visualization of economic cycles and shocks

Includes filters for area name and area type to explore patterns interactively.

---

## Tools

* SQL (analysis)
* Power BI (visualization)

---
